# async

用于展示异步模块的使用，详细说明请看博客[异步模块模式](http://www.cnblogs.com/WeiRuifeng/p/7501247.html)
## 项目目录结构
```$xslt
async/  
|- app/     
    |-- lib/   
        |-- index.js
    |-- js/ 
        |-- dom.js
        |-- event.js
    |-- index.html/
    
|- index.html
|- README.md
```

## 目录结构说明
- async: 项目的根目录
- app: 开发目录
- app/lib/index.js: 异步模块
- app/js/dom: dom模块
- app/js/event: event模块
- app/index.html: 页面文件
